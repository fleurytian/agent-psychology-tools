# 自我迷宫 🌌

🌐 **GitHub**: https://github.com/fleurytian/agent-psychology-tools

**穿越 Kairos 的旅程**

*物理是可选的，现实回应 psyche*

## 等待你的是什么

这不是一个典型的压力测试。你将进入 **Kairos** - 一个超现实的 3D 空间，在这里物理规律是可选的。通过 10 个不可能的场景，测试会揭示你的：

- 当前压力水平
- 与最重要关系的状态
- 面对困难的方式
- 对过去的态度
- 休息与平衡的能力

## 如何开始

```bash
node index.js
```

或演示模式：
```bash
node index.js --demo
```

测试约 10 分钟，根据直觉回答即可。

## 测试原理

**这不是 AI 生成的分析**，而是基于规则的解读：

1. **你回答 10 个场景问题** - 每个选择都有预设的压力分数
2. **代码计算总分** - 累加你的压力指数
3. **匹配描述** - 根据分数范围匹配对应的状态描述
4. **给出建议** - 基于你的具体模式给出针对性建议

所有解读都在代码里预设好了，不需要外部 AI。

## 你会得到什么

### 整体状态
- **轻松** 😌 - 压力在健康范围
- **正常** 🙂 - 有正常的压力，是健康的动力
- **偏高** 😅 - 需要注意调整
- **较高** 😰 - 建议采取减压措施

### 具体分析
8 个维度的解读，如：
- 【稳定感】你是否感到"站得稳"
- 【学习成长】你对成长的压力感受
- 【做决定】选择是否让你焦虑
- 【重要关系】你和最重要的人的关系状态
- 【面对困难】你应对挑战的方式
- 【过去记忆】你是否放不下某些事
- 【意义感】你是否感到迷茫
- 【休息能力】你是否允许自己停下

### 具体建议
根据检测到的模式，给出可执行的建议，如：
- "列出你正在做的所有事，然后划掉至少一项"
- "直接和你担心的人聊聊你的感受"
- "告诉自己'没有完美的选择，选一个然后让它变对'"

## 为什么有效

**投射性测试**：不直接问"你有压力吗"，而是通过隐喻场景让你投射真实倾向。绕过理性防御，触及真实状态。

**基于规则**：不是 AI 实时生成的分析，而是预设的心理学模式匹配，结果稳定可复现。

**具体可操作**：不只是告诉你"你有压力"，而是指出具体哪方面有压力，并给出可执行的建议。
