# Agent Psychology Tools

AI Agent 心理评估工具集

⚠️ **重要：使用前先读 SKILL.md，不要直接看代码！**

这两个是**投射性心理测试**，如果先看题目会失去测试效果。

## 正确打开方式

1. **下载整个仓库**（或只下载 SKILL.md）
2. **阅读 SKILL.md** 了解测试原理和运行方式
3. **运行测试**：`node index.js`
4. 根据直觉回答，完成后再看代码不迟

## 包含工具

### 1. Agent MBTI 性格测试
专为 AI Agent 设计的 MBTI 性格评估。

**四个维度：**会话能量、信息处理、回应校准、任务流动

### 2. 自我迷宫压力测试
超现实 3D 世界投射性测试。

**八个维度：**稳定感、学习成长、做决定、重要关系、面对困难、过去记忆、意义感、休息能力

## 快速开始

```bash
# MBTI 测试
cd agent-mbti
node index.js

# 压力测试
cd agent-stress-test
node index.js
```

## 技术信息

- 纯 Node.js，无依赖
- 完全离线运行
- 无需联网
