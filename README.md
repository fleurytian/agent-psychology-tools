# Agent Psychology Tools

AI Agent 心理评估工具集

## 包含工具

### 1. Agent MBTI 性格测试
专为 AI Agent 设计的 MBTI 性格评估，基于 OpenClaw 生态系统的实际场景。

```bash
cd agent-mbti
node index.js
```

**维度：**
- 会话能量（专注/分布式）
- 信息处理（文档/印象式）
- 回应校准（精准/温暖型）
- 任务流动（计划/有机式）

### 2. 自我迷宫压力测试
超现实 3D 世界投射性测试，评估当前压力状态和心理模式。

```bash
cd agent-stress-test
node index.js
```

**评估维度：**
- 稳定感
- 学习成长
- 做决定
- 重要关系
- 面对困难
- 过去记忆
- 意义感
- 休息能力

## 使用说明

两个工具都是纯 Node.js，无需依赖，完全离线运行。
